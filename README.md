### 라이브러리 다운로드
```
npm install @toast-ui/react-editor @toast-ui/editor
```
위의 명령어로 install이 되지 않는 경우
```
yarn add @toast-ui/react-editor @toast-ui/editor
```
위의 명령어로 install(저는 yarn으로 했어용..)

### 설치 확인! (pakage.json)
```
"dependencies": {
  "@toast-ui/editor": "^3.x.x",
  "@toast-ui/react-editor": "^3.x.x"
}
```

### 실행
```
npm start 
```
```
yarn start
```
